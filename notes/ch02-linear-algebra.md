## 一、特征值与特征向量 — 从定义到直觉

### 1.1 数学定义

```
A·v = λ·v

A: n×n矩阵
v: 特征向量 ≠ 0
λ: 特征值（可以是复数）
```

**直觉翻译**：矩阵 A 作用在 v 上，等于把 v 拉长/缩短 λ 倍。v 的方向不变，只被拉伸。

### 1.2 为什么对称矩阵对所有量化计算都很关键？

MIT讲义命题3：

> 对称矩阵 A = Aᵀ 的两个性质：
> (i) 所有特征值都是实数
> (ii) 可以正交对角化：A = UDUᵀ，U的列相互正交且为单位长度

**为什么重要**：
- 协方差矩阵天然对称 → 特征值天然实数 → 代表真实波动能量
- 正交对角化 = 可以用旋转代替矩阵运算 → 计算效率
- 这在组合优化中有决定性意义

### 1.3 最大特征值的变分性质

定理4：
```
max_{||v||=1} |Av| = max_λ |λ|
```

翻译：一个矩阵作用在单位向量上能产生的最大拉伸量 = 最大特征值。

**你的系统里**：网格的最大风险方向 = 协方差矩阵的最大特征值对应的特征向量（即第一主成分）。这个方向上的资产组合风险最大。

## 二、奇异值分解 (SVD) — 通用矩阵分解

### 2.1 与特征值分解的对比

| | 特征值分解 | SVD |
|---|---|---|
| 适用矩阵 | 方阵 (n×n)，可对角化 | 任意矩阵 (m×n) |
| 分解形式 | A = PDP⁻¹ | A = UΣVᵀ |
| U的含义 | 特征向量 | 左奇异向量 (Rm空间) |
| Σ的含义 | 特征值 | 奇异值 (≥0, 降序) |
| V的含义 | — | 右奇异向量 (Rn空间) |
| 对称矩阵 | A = UDUᵀ (正交对角化) | A = UΣVᵀ (Σ为特征值的绝对值) |

### 2.2 SVD的构造（证明中的关键步骤）

```
1. 计算 AᵀA → n×n 对称半正定矩阵
2. AᵀA的非零特征值 = σ₁², σ₂², ..., σᵣ² （σᵢ > 0 是奇异值）
3. AᵀA的特征向量 → V矩阵的列
4. uᵢ = A·vᵢ / σᵢ → U矩阵的列（需补成完整正交基）
5. A = U·Σ·Vᵀ
```

### 2.3 简化SVD（m ≤ n 时）

当行数 ≤ 列数（资产数 ≤ 时间点数），可以只用m×m的Σ：
```
A(m×n) = U(m×m) · Σ(m×m) · V(n×m)ᵀ
```

**你的数据**：通常资产数(m) << 时间点数(n)。a-stock-data可以取100只股票(m=100)，每只300天(n=300)。简化SVD就能用：100×100的U和Σ。

### 2.4 SVD 在量化中的核心应用

| 应用 | 矩阵 | 分解含义 |
|------|------|---------|
| PCA主成分分析 | 收益率矩阵 R(m×n) | 第一PC = 市场因子, 第二PC = 行业因子 |
| 因子模型构建 | R ≈ UₖΣₖVₖᵀ (k-SVD) | 低秩近似 = 用k个因子解释收益 |
| 协方差估计 | Σ = AᵀA/n | 特征值 = 各风险来源的能量 |
| 风险预算 | UΣ¹/² = 风险分解 | 把组合风险归因到各因子 |
| 降噪 | 截断k个最大奇异值 | 去噪声 → 更稳定的协方差矩阵 |

## 三、Perron-Frobenius定理

### 3.1 定理内容

> 如果矩阵 A 的所有元素 > 0（全部正数），则：
> 1. 存在一个正实数特征值 λ₀，所有其他特征值满足 |λ| < λ₀
> 2. 对应 λ₀ 的特征向量所有分量 > 0
> 3. λ₀ 是单根（唯一）

### 3.2 金融含义

这解释了为什么**市值加权的指数（如沪深300）天然具有稳定性**：

- 市值权重矩阵 = 所有正数 → Perron-Frobenius定理适用
- 最大特征值对应的特征向量 ≈ 市值权重向量
- 所有其他"模式"（板块轮动、风格轮动）的特征值都小于λ₀
- 这意味着**市值因子是不可消除的主导因子**

对你的系统：如果持仓的β对市值因子暴露过高，任何网格策略都会被市值因子的波动主导。

## 四、金融实例：股票价格矩阵的SVD

### 4.1 MIT讲义中的例子

```
10×10矩阵：行=10家公司, 列=10个日期
(i,j)格 = 第i家公司第j天的股价
```

问题：这个矩阵代表什么"变换"？

答案：SVD给出：
- U的列 = 10个"价格模式"（在股票空间）
- V的列 = 10个"时间模式"（在时间空间）
- Σ的奇异值 = 每个模式的强度

第一个奇异值通常远大于其他的 → 第一PC = "市场模式"（所有股票一起涨跌）。第二个PC可能 = "大小盘轮动"。第三个可能 = "行业差异"。

## 🔥 实践对照：与量化系统的映射

### 对照1：SVD → 主成分分析 (PCA)

```
你的系统：a-stock-data可以批量获取100只股票的300天日K线
→ 构建收益率矩阵 R(100×300)
→ R的每一列减去列均值（中心化）
→ SVD分解: R_centered = U·Σ·Vᵀ
→ 前k个奇异值占比 Σᵢ²/Σᵢ² = 方差解释率
→ 前k个PC作为因子
→ 用这些因子做因子中性化 → 消除beta后看alpha
```

**现在可以做的事**：
- 用PCA提取沪深300的隐含因子（市场/行业/市值等）
- 对你的持仓做因子暴露归因
- 因子中性化后看是否还有alpha

### 对照2：对称矩阵 → 协方差矩阵的特征结构

```
你的网格同时持有BTC和ETH
→ 构建2×2协方差矩阵
→ 特征值λ₁>λ₂对应特征向量v₁, v₂
→ v₁ = 最大风险方向（BTC和ETH联动的方向）
→ λ₁/(λ₁+λ₂) = 该方向的风险占比
→ 如果此比例>95% → BTC和ETH本质上是一个篮子
```

这对你的bet_8（多标的网格：BTC+ETH+SOL）极其重要：如果协方差矩阵的"第一特征值占比"很高，那多标的不等于分散化。

### 对照3：Perron-Frobenius → 你的因子中性化问题

```
Perron-Frobenius定理告诉你：所有正数的矩阵有主导特征值
这意味着：市值因子永远存在，不可消除

如果不对市值做中性化 → Beta分析被市值主导
如果中性化 → Beta更有意义（衡量的是纯行业/风格的Beta）
```
