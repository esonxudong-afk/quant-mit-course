## 一、随机游走 — BTC价格到底是不是随机游走？

### 1.1 定义

```
Y₁, Y₂, ...  i.i.d.  P(Y=+1)=P(Y=-1)=0.5
X₀ = 0
X_k = Y₁ + Y₂ + ... + Y_k  (求和)
```

### 1.2 三大性质

| 性质 | 含义 | 对网格交易的影响 |
|------|------|----------------|
| E[X_k] = 0 | 长期期望为零 | 如果价格是纯随机游走，网格长期不赚钱 |
| 独立增量 | 过去不影响未来步长 | 止损后不会"反弹", 每次都是独立事件 |
| 平稳增量 | 步长的分布在时间上不变 | 波动率不变假设是错的 |

**CLT 推论**：对于足够大的 k，√(1/k)·X_k → N(0,1)。

### 1.3 随机游走的均值回复 vs 趋势 — 赌徒博弈

```
A 和 B 是两个边界
P(到达A之前到达B) = B / (A+B)

从 k=0 出发，到 A=5 之前到 B=5 的概率 = 5/10 = 50%
```

**这直接关系到你的网格**：上下沿 = A 和 -B 边界。如果价格是纯随机游走，**网格上下沿触达概率完全对称** — 这解释了为什么涨跌都有可能打穿网格。

但 MIT 隐含的问题是**如果它不是纯随机游走呢？**

- 如果有一个细微的趋势（μ≠0），可选停止定理说长时间后优势方必胜
- 如果有均值回复（自相关），到达边界的概率就不再是 B/(A+B)
- **你的网格赚钱的前提是：价格不是纯随机游走，而是有均值回复的**

## 二、马尔可夫链 — 市场状态切换

### 2.1 定义

```
P(X_{n+1} = j | X_n, X_{n-1}, ..., X_0) = P(X_{n+1} = j | X_n)
```

未来只依赖于现在，不依赖于过去。这是量化策略的最强假设。

### 2.2 转移概率矩阵

```
A = [p_ij] where p_ij = P(X_{n+1}=j | X_n=i)
```

**MIT 关键定理**：如果所有 p_ij > 0，存在唯一平稳分布 π，且不管从哪开始，长期比例都收敛到 π。

```
lim_{n→∞} r_ij(n) = π_j
```

π 是 A 的特征向量，对应特征值 1。**这和第二章讲的 Perron-Frobenius 定理直接相连**。

### 2.3 你的 regime 检测 — 本质是马尔可夫状态识别

```
你的 ADX/波动率分位 → 把市场分成两种状态:
  状态1: 震荡 (适合网格)
  状态2: 趋势 (不适合网格)

隐含假设: 状态切换是马尔可夫的
  P(明天趋势|今天是震荡)=某个概率
  P(明天震荡|今天是震荡)=某个概率
```

**问题**：你当前的 bet_runner 用的是**绝对阈值**（ADX > 35 = 趋势），但 MIT 告诉你应该用**转移概率**。

阈值是确定性的 → 误报率高
转移概率是概率性的 → 更平滑，更稳健

### 2.4 平稳分布告诉你什么

```
如果从你的数据中估计出:
A = [[0.7, 0.3],   # 震荡→震荡70%, 震荡→趋势30%
     [0.4, 0.6]]   # 趋势→震荡40%, 趋势→趋势60%

稳态分布 π_震荡 ≈ 57%, π_趋势 ≈ 43%

→ 长期来看, 57%的时间适合网格, 43%不适合
→ 如果你的回测在牛市(趋势占比更高), 可能高估了网格效果
```

## 三、鞅 — 判断你的策略是 alpha 还是运气

### 3.1 定义

```
X_t = E[X_{t+1} | X_0, X_1, ..., X_t]
```

翻译：**在已知所有历史信息的情况下，下一期的条件期望等于当前值**。这是"公平博弈"的数学定义。

### 3.2 你的网格检验 — 净值是不是鞅？

```
假设净值序列 V_0, V_1, ..., V_n

如果 V 是鞅 → E[V_{t+1} | 历史] = V_t
  → 你在任何时刻的期望收益都是零
  → 网格没有 alpha

如果 V 是下鞅 → E[V_{t+1} | 历史] > V_t
  → 你有正期望, 策略有 alpha

如果 V 是上鞅 → E[V_{t+1} | 历史] < V_t
  → 你在亏钱
```

**检验方法**：对净值序列做方差比检验 (Variance Ratio Test)

```
VR(k) = Var(r_t + r_{t-1} + ... + r_{t-k+1}) / (k · Var(r_t))

如果价格是随机游走 → VR ≈ 1
如果 VR < 1 → 均值回复 → 网格有优势
如果 VR > 1 → 趋势/动量 → 网格的敌人
```

## 四、可选停止定理 — 就算有止损，公平博弈下也不会赚钱

### 4.1 定理

```
如果 τ 是有界停止时间（τ ≤ T），且 X 是鞅
则 E[X_τ] = E[X_0]
```

### 4.2 对你的系统的残酷含义

```
你的网格有:
  止盈线: 价格到达网格上沿 → 平仓
  止损线: 价格突破网格下沿 → 止损

如果价格是纯随机游走(鞅):
  E[止盈时的净值] = E[止损时的净值] = 初始净值
  → 止盈止损不会改变你的期望收益!

你赚钱的唯一前提:
  价格不是鞅 — 存在均值回复 → 止盈概率 > 止损概率
```

**这就是为什么你的 bet_4 和 bet_5 是 NOGO** — 如果在回测中没有检测到均值回复（ADF 检验、VR 检验都不拒绝随机游走），网格不可能赚钱。

### 4.3 有优势时 — 优势多小都能赢

MIT 讲义的最后一段：
> "if one has some advantage over an opponent in some game, then no matter how small that advantage is, he/she will win in the long run."

这是对赌场的描述（赌场优势 ~1-5%）。同样的逻辑适用于你的网格 — 只要均值回复真实存在（哪怕只有 0.5%），长期你就能赢。关键是**你回测的数据里能不能证实这个优势**。
