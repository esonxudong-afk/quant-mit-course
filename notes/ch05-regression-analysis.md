## 一、OLS — 你已经用了但没完全理解

### 1.1 正规方程

```
β̂ = (XᵀX)⁻¹Xᵀy
```

设计矩阵 X 的列满秩 → XᵀX 可逆 → β̂ 唯一。这和第二章的 SVD 直接相连：XᵀX 的逆可以用 SVD 稳定求解（不用求逆）。

### 1.2 Hat Matrix H = X(XᵀX)⁻¹Xᵀ

```
ŷ = Hy    (预测值 = H矩阵 × 真实值)
ê = (I-H)y    (残差 = 去除被模型解释部分后的剩余)
```

H 把 y 投影到 X 的列空间上。**H 的迹 = p（参数个数）**，这是模型复杂度的无偏估计。

## 二、Gauss-Markov — OLS为什么最优

### 2.1 定理

在高斯-马尔可夫假设下（误差均值0、方差恒定σ²、不相关），**OLS λ̂ 是所有线性无偏估计中方差最小的**（BLUE = Best Linear Unbiased Estimator）。

### 2.2 三个前提 — 你的数据满足吗？

| 假设 | 含义 | BTC/ETH/SOL 1474天 | 后果 |
|------|------|---------------------|------|
| E[ε]=0 | 模型正确设定 | ❌ 第三章正态性不成立 | β̂仍无偏但不是最优 |
| Var(ε)=σ² | 同方差 | ⚠️ 需检验 | 异方差 → SE(β̂)偏低 → 假阳性 |
| Cov(εᵢ,εⱼ)=0 | 不相关 | ❌ 第四章自相关存在 | SE(β̂)偏小 → t值偏高 |

**结论：你第一章的beta_alpha_monitor假设了Gauss-Markov，但数据不满足后两条。这与第三章的非正态发现一致。**

## 三、GLS — 处理异方差和自相关

```
GLS: β̂ = (XᵀΣ⁻¹X)⁻¹XᵀΣ⁻¹y
```

如果误差协方差矩阵 Σ ≠ σ²I，OLS 不再是最优的。GLS 用 Σ⁻¹ 加权——误差大的观测给低权重。

**这对你的意义**：如果 BTC 不同时间段的波动率差异巨大（牛市波动 vs 熊市波动），GLS 能让最近的数据给更高权重。

---

## 四、正态回归模型 — t检验的理论基础

```
正态假设下:
β̂ⱼ ∼ N(βⱼ, σ²·Cⱼⱼ)  其中 C = (XᵀX)⁻¹
t = (β̂ⱼ - βⱼ) / (σ̂·√Cⱼⱼ) ∼ t_{n-p}
```

这就是第一章 T 检验的数学来源。

**矩阵版本**：
```
β̂ ∼ N_p(β, σ²(XᵀX)⁻¹)
```

## 五、QR分解 — 更稳的OLS求解

```
X = QR  (Q: n×p正交, R: p×p上三角)
β̂ = R⁻¹Qᵀy   (不需要求XᵀX的逆, 更稳定)
```

当 X 接近共线时（你的BTC/ETH/SOL就是），XᵀX 的逆数值不稳定。QR 分解（通过 Gram-Schmidt）更稳。numpy 的 `np.linalg.lstsq` 内部就用 QR 或 SVD。

## 六、MLE — OLS的概率版本

```
假设 ε ∼ N(0,σ²):
ML的β̂ = OLS的β̂ (两者一致)
MLE的σ² = RSS/n (有偏，n-p才是无偏)
```

---

## 七、稳健M估计 — 处理异常值

```
OLS: min Σ(yᵢ - xᵢᵀβ)²
M-估计: min Σ χ(yᵢ - xᵢᵀβ)  (χ是稳健损失函数)

Huber损失: χ(u) = u²/2 if |u|≤k, k|u|-k²/2 if |u|>k
```

**第三章发现你的收益率是肥尾的**。肥尾 → 存在大量"异常值" → OLS 被异常值拉偏（OLS 对大残差平方非常敏感）→ 稳健M估计更可靠。

### 对照1: 异方差检验

```
BTC 1474天日收益率: 
早期(2022) σ≈40%, 近期(2024-2026) σ≈50%
→ 方差不是常数 → Gauss-Markov 违反
→ OLS的SE(β̂)不准确
→ 用 White's heteroskedasticity-consistent SE 修正
```

### 对照2: θ=0 vs OLS

```

→ 但 var(β̂) 被低估 → t统计量偏高 → 容易得出假阳性

正确做法: Huber-White heteroskedasticity-consistent covariance
```
