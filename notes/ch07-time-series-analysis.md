## 核心：平稳性 → ARMA → ARIMA → ADF检验

### 一、协方差平稳

```
E(X_t) = μ (常数)
Var(X_t) = σ² (常数)
Cov(X_t, X_{t+τ}) = γ(τ) (只取决于间隔τ)
```

**BTC/ETH/SOL 1474天数据**：第四章VR检验不拒绝随机游走 → 价格不平稳（一阶差分后才平稳）。这意味着：**需要用ARIMA而非ARMA建模**。

### 二、Wold定理 — 任何平稳序列都是无限阶MA

```
X_t = V_t + Σᵢ₌₀∞ ψᵢ·η_{t-i}
ψ₀=1, Σψᵢ²<∞, η_t是白噪声
```

### 三、ARMA(p,q)

```
X_t = φ₁X_{t-1}+···+φ_pX_{t-p} + ε_t + θ₁ε_{t-1}+···+θ_qε_{t-q}
```

PACF截断 → 选AR阶数p；ACF截断 → 选MA阶数q。

### 四、ARIMA(p,d,q) — 差分后平稳

d=1: ΔX_t = X_t - X_{t-1} → 对差分序列建模

### 五、ADF检验 — 判断是否需要差分

```
H0: 有单位根(不平稳) → d≥1
H1: 没有单位根(平稳) → d=0
```
