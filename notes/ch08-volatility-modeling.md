## 核心：波动率预测 — GARCH

### 一、历史波动率方法

```
σ̂ = sqrt(1/(T-1) Σ(R_t - R̄)²)
年化: √252 × σ̂_daily

预测: 简单MA, EWMA, RiskMetrics
EWMA: σ̃²_{t+1} = (1-λ)σ̂²_t + λ·σ̃²_t
```

### 二、GARCH(p,q) — 波动率聚集

```
ARCH(q): σ²_t = ω + Σα_i·ε²_{t-i}
GARCH(p,q): σ²_t = ω + Σα_i·ε²_{t-i} + Σβ_j·σ²_{t-j}
GARCH(1,1): σ²_t = ω + α·ε²_{t-1} + β·σ²_{t-1}
   需要 α+β < 1 (平稳性)
```

### 三、波动率聚集 → 对你的网格的含义

第二章 dynamic_spread 用了固定回望窗口标准差。但如果波动率有聚集性（大波动→大波动），应该用EWMA或GARCH预测的σ，而不是等权标准差。
